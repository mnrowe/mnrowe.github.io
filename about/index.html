Timestamp Microservice

Created a small API, you can pass it either a unix timestamp or date. It returns both a unix timestamp and date in the form of a JSON object. If neither a unix timestamp or date is passed it returns null.

Courses

CS50 | 1 Module Left

A MOOC course offered by Harvard University it’s an introductory course to computer science. It stats off teaching scratch and giving you the basics of what programming languages are like. It start to move towards C and gives you an in-depth explanations of basic algorithms and tools to measure how effective algorithms are in certain scenarios like Big O notation. It also dives deep into memory management the memory stack, heap, buffer overflows, and pointers. The later parts of the course move on to higher level languages like python and building full stack applications with python.

freeCodeCamp

This course starts you out on the basics of web development using Javascript, HTML, and CSS. It tries to build you up to solving easier algorithms using javascript. Eventually it leads to building projects like calculators, pomodoro clocks, and Markdown Previewers.
